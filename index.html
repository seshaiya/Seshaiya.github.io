Medicare Advantage Risk Adjustment & Clinical Compliance Dashboard

May 2022 - Dec 2025

Client: Optum – United Health Care (UHC)

United Health Care (UHC), a part of Optum, manages large Medicare Advantage populations requiring accurate risk adjustment coding and CMS compliance. Accurate HCC (Hierarchical Condition Category) and ICD-10 documentation directly impact RAF (Risk Adjustment Factor) scores, regulatory reporting, and reimbursement accuracy. The objective of this project was to design an interactive Power BI dashboard to monitor HCC capture rates, ICD-10 coding accuracy, documentation gaps, and CMS compliance performance. The solution supported audit teams, clinical documentation improvement (CDI) teams, and risk adjustment leadership in improving coding quality, identifying missed diagnoses, and enhancing predictive risk modeling initiatives.

Roles and Responsibilities:

  • Connected Power BI to structured healthcare data sources including SQL Server databases, EMR extracts, Excel audit trackers, and risk adjustment datasets to consolidate patient diagnosis and coding information.
  • Performed data cleansing and transformation using Power Query Editor, including: Standardizing ICD-10 codes, Mapping ICD-10 codes to HCC categories, Cleaning duplicate patient encounter records, Creating derived columns for RAF calculation and compliance indicators.
  • Built interactive dashboards using trend visuals, compliance scorecards, matrix reports, drill-through pages, and slicers to analyze patient risk distribution and provider performance.
Tech: Power BI · SQL Server · Power Query Editor · Excel · EMR Data
